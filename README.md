# Hill Fill Phil

Get Phil up the hill, by filling your hard drive.

Made over 72 hours (mostly at CampJS IX) for Ludum Dare 42.

## Develop

```sh
npm install
npm start
```

## Build

```sh
npm run dist -- -m # macOS
npm run dist -- -w # Windows
npm run dist -- -l # Linux
```
